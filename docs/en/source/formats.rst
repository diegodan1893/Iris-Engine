Supported formats
=================

Here you will find a list of the file formats supported by Iris Engine.


Audio file formats
^^^^^^^^^^^^^^^^^^

* WAV
* FLAC
* OGG


Image file formats
^^^^^^^^^^^^^^^^^^

* JPG
* PNG
* BMP
  

Font file formats
^^^^^^^^^^^^^^^^^

* TTF
* OTF
  

Video file formats
^^^^^^^^^^^^^^^^^^

* OGV (audio bitrate must be 44100 kHz)
